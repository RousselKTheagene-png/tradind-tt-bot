Add indicators (EMA/BB/RSI) and candlestick pattern detection to chart

- src/dashboard/app.py:
  - _fetch_candles(): cached OHLCV helper shared by /chart, /indicators, /patterns
  - GET /indicators: EMA20, EMA50, Bollinger (20,2) upper/mid/lower, RSI(14)
  - GET /patterns: hammer, shooting star, bullish/bearish engulfing, doji
    (wick-to-body and wick/range ratios; hammer/star evaluated before doji so
    small-body reversal candles are not shadowed)
- src/dashboard/html.py:
  - Indicator toggle row (EMA 20 / EMA 50 / BB / RSI / Patterns)
  - Multiple LineSeries for overlays, dedicated RSI sub-pane that shows only
    when toggled on
  - patternMarkers builder renders circles with short labels (H / SS / BE / D)
  - Signals and patterns merged into one marker layer, bucketed to timeframe
- tests/test_dashboard.py: 3 new tests (hammer, bull engulfing, doji);
  suite now 18/18 passing

Live verification on Kraken BTC/USDT 1h x 200:

  patterns:   64 found (mix of all 5 types)
